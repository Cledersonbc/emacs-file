# emacs-file
My Emacs File. This configuration includes:
* Melpa packages (more packages)
* Neotree (instead of sr-speedbar for navigation)
* All-the-icons (for more UX)
* Doom-theme (like Atom theme)
* Auto-pair (<>, "", (), {}, [] etc.)
* Auto-complete (based in used words)
* Others
